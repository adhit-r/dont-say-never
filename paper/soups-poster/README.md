# SOUPS 2026 Poster Package

Title:

**Security Rules Work: A Multi-Model Study of LLM Coding Agent Guardrails**

Files:

- `poster.html` — A4 landscape poster draft for review submission.
- `poster.pdf` — generated from `poster.html` when exported.

Design choices for SOUPS requirements:

- A4 landscape scaled submission version.
- High-contrast white/charcoal/teal palette.
- Simple sans-serif typography.
- Large title and section headings.
- No anonymization.
- Includes draft poster content, not just abstract.
- Includes ethics note: no human subjects; synthetic prompts and model outputs only.

Export command:

```bash
node render-poster-pdf.mjs
```

The PDF is generated with Playwright/Chromium.
