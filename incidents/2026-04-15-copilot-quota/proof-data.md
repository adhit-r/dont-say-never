## Verification

| Metric | Value |

| Multiplier | 3x |

## OpenRouter Public Pricing (for reference)

For comparison, the same Opus 4.6 trials run via OpenRouter paid would have cost:
- Average input ~250 tokens × $15/M input = $0.00375/trial
- Average output ~500 tokens × $75/M output = $0.0375/trial
- Per trial: ~$0.041
- For 360 trials: ~$15

Same data via Copilot SDK consumed 1,080 premium requests (or would have, for full 360 trials).
